# 7 Claude Code Skills Pack

Custom skills I use daily. Drop into ~/.claude/skills/ and they work instantly.

| Skill | What it does |
|-------|-------------|
| debug | Systematic debugging: reproduce, isolate, root cause, fix, verify |
| review | 4-pass code review: implementer, debugger, security, verification |
| audit | Full security + quality + performance audit |
| bulletproof | 5-phase protocol for critical code paths |
| code | Disciplined implementation: read first, trace deps, handle errors |
| orchestrate | Multi-agent coordination with parallel workstreams |
| tokenopt | Token optimization for prompts and LLM app code |

## Install

```bash
mkdir -p ~/.claude/skills
cp *.md ~/.claude/skills/
```
